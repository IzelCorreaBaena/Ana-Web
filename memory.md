## PROYECTO
- Nombre: AnaCastellano
- Objetivo: Web profesional + panel admin para "Ana Castellano Florista"
- Estado: CREATE

## STACK
- Lenguaje: TypeScript
- Framework Frontend: Vite + React + TailwindCSS
- Framework Backend: Node.js + Express + TypeScript
- Base de datos: PostgreSQL + Prisma ORM
- Auth: JWT
- Email: Nodemailer
- Integración: Google Calendar API

## SECCIONES WEB
1. Inicio - presentación, CTA
2. Sobre Ella - historia, proyectos, reseñas
3. Servicios - bloques reutilizables
4. Contacto - teléfono, email, redes, formulario
5. Reservas - formulario (nombre, teléfono, email, servicio)

## PANEL ADMIN
- Login seguro (JWT)
- Gestión servicios (CRUD bloques: texto + imágenes)
- Gestión reservas (ver, aceptar/rechazar)
- Aceptar: email confirmación + Google Calendar
- Rechazar: email cancelación

## TABLAS DB
- admin
- servicios
- bloques
- reservas
